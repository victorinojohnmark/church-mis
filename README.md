## CHURCH MIS
Capstone project

## Dev Requirements
<ul>
    <li>PHP 8.1+</li>
    <li>MySQL</li>
    <li>Apache 2.4</li>
    <li>Node Package Manager or NodeJS</li>
    <li>Composer</li>
</ul>

## Project Setup

```sh
composer install
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
